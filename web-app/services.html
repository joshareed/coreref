Resources

/services/track/{collection}/{type}/{top}/{base}/{scale}

This service renders tracks of imagery and data as PNG images.

resource-wide template parameters
parametervaluedescription

collection

string (required)

One of:

  • and1-1b
  • iodp-0807c

type

string (required)

One of:

  • split
  • whole
  • lith
  • ruler
split

Split core image track

whole

Whole core image track

lith

Lithology track

ruler

Ruler track

top

double (required)

The top depth in collection-specific depth units, usually meters (below sea floor). Example: 100.25

base

double

Default: top + 1.0

The base depth in collection-specific depth units, usually meters (below sea floor). It is assumed that this value will be greater than that for top. Example: 101.25

scale

int

Default: 2000

The scale in pixels per depth unit to render the track at. Example: 2000

Methods

GET

available response representations:

  • image/png

Representations

image/png
